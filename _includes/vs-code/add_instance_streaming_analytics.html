If you already have a Streaming Analytics service in IBM Cloud, make sure that it is started and running. If you need to create a Streaming Analytics service, follow these steps:

  • Go to IBM Cloud and sign in (or sign up for a free account).
  • Go to the Streaming Analytics service page within the Catalog.
  • Enter a service name (or accept the default name) and then click Create to set up your service. You will be redirected to the service list and the new service will be listed under Services. It will take some time for the service to be provisioned. Once it finishes, click on the service to see the service details page. Your service is started automatically.

To add an instance in VS Code, either:

  • Bring up the Streams Explorer view by clicking on the Streams icon on the left side of VS Code. Then, click on the + button.
  • Run the IBM Streams: Add Instance command.

You must provide the following information in order for the Streams extension to connect to your service.

  • Service credentials (in JSON format): You may add or view your credentials in your service details page. Refer to the documentation for more details.

Once you add the instance, you can view the instance by clicking on the IBM Streams icon on the left side to bring up the Streams Explorer view.
